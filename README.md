# hobby matching

Discord などのコミュニティで、誰とどんな話題が合うのかを見つけやすくする趣味マッチングアプリです。  
趣味プロフィールと共通点の可視化によって、会話のきっかけを自然に作れる体験を目指して開発しました。

## サービス概要

オンラインコミュニティでは、同じ場所にいても

- 誰が何の趣味を持っているのか分かりにくい
- 共通の話題がある相手を見つけるまでに時間がかかる
- 初対面や久しぶりの相手には話しかけるハードルが高い

といった課題があります。

`hobby matching` は、ユーザーごとの趣味をプロフィールとして登録し、同じコミュニティ内のメンバー同士の共通点を可視化することで、「誰と何を話せるか」を見つけやすくする補助ツールです。

---

## 解決したい課題

### 1. 共通の話題が見つけにくい

SNS やチャットでは、プロフィール文や過去投稿を読まないと相手の趣味が分からず、会話のきっかけを探すコストが高くなりがちです。

### 2. コミュニティ内の関係性が見えにくい

人数が増えるほど、「誰と誰が似た興味を持っているか」を把握するのが難しくなります。

### 3. 話しかける心理的ハードルが高い

共通点が見えない状態では、「何を話せばいいのか分からない」が先に立ち、交流が始まりにくくなります。

---

## 解決方法

このアプリでは、上記の課題に対して次のようにアプローチしています。

- 趣味をタグ形式で登録し、プロフィールとして一覧化する
- 他ユーザーのプロフィールを横断的に見られるようにする
- 共通の趣味をプロフィール上で表示し、会話の入口を明確にする
- 部屋単位でメンバーを管理し、マインドマップで共通点を可視化する
- 招待リンクを使って、コミュニティ単位で参加メンバーを集められるようにする

単なる趣味管理ではなく、会話が始まる前の迷いを減らすことに重きを置いています。

## 主要機能

| 機能 | 内容 |
| --- | --- |
| 1. 趣味プロフィール作成 | アバター、自己紹介、趣味タグを登録<br>各趣味ごとに説明文を持たせて、好きな理由や温度感も表現可能 |
| 2. プロフィール一覧・詳細表示 | 登録済みユーザーのプロフィールを一覧で確認<br>他ユーザー詳細画面で共通の趣味を表示 |
| 3. 趣味タグの入力補助 | オートコンプリートにより既存タグを再利用しやすい設計<br>表記ゆれや重複登録を抑えながら入力できるように工夫 |
| 4. 部屋作成・招待リンク共有 | コミュニティごとに部屋を作成<br>公開 / ロック状態や招待リンクの有効期限を設定<br>招待リンク経由で参加メンバーを集約 |
| 5. マインドマップでの共通趣味可視化 | 部屋に参加したメンバー同士の共通点をマインドマップで表示<br>「誰とどの趣味が重なっているか」を視覚的に把握可能 |
| 6. 管理者による趣味整理 | 親タグ管理<br>未分類趣味の分類・統合 |

ユーザーが入力した趣味データを育てながら、検索性と見やすさを保てる構成にしています。

## 工夫したポイント

### 会話のきっかけづくりにフォーカスした設計

趣味を保存すること自体ではなく、「次に誰へ話しかけるか」が見えることを価値として設計しました。  
そのため、プロフィール登録だけで終わらず、共通趣味の表示や部屋単位での可視化までつなげています。

### テキスト情報を視覚情報に変換

プロフィールの趣味データをそのまま並べるだけではなく、マインドマップとして見せることで、メンバー間のつながりを直感的に把握しやすくしました。

### コミュニティ利用を意識した運用設計

1対1のマッチングではなく、部屋単位でメンバーを集めて共通点を見られるようにし、Discord のような複数人コミュニティで使いやすい形を意識しています。

### 入力データを管理しやすい構成

趣味は自由入力のままだと増殖しやすいため、管理画面で未分類タグの整理や統合ができるようにして、継続利用に耐える設計を意識しました。
